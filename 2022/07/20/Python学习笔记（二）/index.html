Python 运算符详解

成员运算符 (in, not in)

基本用法

用于判断元素是否存在于某个容器中

注意事项

  1. 字典的 in 操作默认检查 key 不是 value
  2. not in 是推荐的否定形式，比 not ... in ... 更清晰

身份运算符 (is, is not)

基本用法

用于比较两个对象的 内存地址 是否相同（即是否是同一个对象）

与 == 的区别

运算符 比较内容 示例
is 内存地址 x is y
== 值相等 x == y

使用场景

  1. 判断 None 时推荐使用 is

  2. 检查布尔值时更清晰

总结表格

运算符类型 运算符 作用 适用容器
成员运算符 in 判断元素是否存在 str/list/dict/set/tuple
not in 判断元素不存在 同上
身份运算符 is 判断是同一对象 所有对象
is not 判断不是同一对象 所有对象
