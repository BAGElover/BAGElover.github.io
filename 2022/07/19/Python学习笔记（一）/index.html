Python 基础概念梳理

可变类型与不可变类型

可变类型 (Mutable)

  • 列表 (list)
  • 字典 (dict)

不可变类型 (Immutable)

  • 整型 (int)
  • 浮点型 (float)
  • 布尔型 (bool)
  • 字符串 (str)

📌 注意：可变类型可以在不创建新对象的情况下修改其内容，而不可变类型的值一旦创建就不能改变。

布尔值 (Boolean)

特点

  • 只有两个值：True 和 False

  • 为 False 的情况：

    • 0（数字零）
    • None
    • 空字符串 ""
    • 空列表 []
    • 空字典 {}
    • 空元组 ()

  • 其他所有值在布尔上下文中都被视为 True

  3. or - 逻辑或
